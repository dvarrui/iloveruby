PSeudo Intérprete


Bienvenido al sistema de ayuda de PSeInt. Esta ayuda se organiza según las siguientes secciones:

  • Interfaz: Describe los principales elementos de la interfaz (botones, menús, cuadros de diálogo, etc).
  • Pseudocódigo: Aquí se describen las principales características del pseudocódigo interpretado, sus primitivas, estructuras, etc.
  • Ejemplos: Contiene una enumeración de los ejemplos de algoritmos incluidos con el programa y sus correspondientes descripciones.
  • Seguimiento: Introduce brevemente la ejecución paso a paso para el seguimiento de algoritmos y la realización de pruebas de escritorio automáticas.
  • Información para docentes: Se describen acciones que normalmente solo deberán realizar los docentes, pero no los alumnos.
  • Notas sobre PSeInt: licencia, registro de cambios, información de contacto, etc.


Puedes buscar rápidamente en la ayuda utilizando el cuadro de texto ubicado en la parte superior de esta ventana.

Tip: Puedes acceder a la ayuda de una palabra clave en la ventana principal desde el pseudocódigo posicionando el cursor de texto sobre la misma y presionando Shift+F1.

Si no encuentras lo que buscas, puedes acceder mediante este enlace a foros online para hacer tu pregunta.
