calcular(index,expresion,traza)

Devuelve un String con el resultado de evaluar el operador expresion[index]. Debemos tener en cuenta que:

  1. index es un valor numérico que identifica un elemento del vector expresión.
  2. expresion es un vector que contiene como elementos cada uno de los componentes de la expresión.
  3. traza es un vector que contiene un registro de las operaciones realizadas.
numero?(p)

Devuelve true si el parámetro es un String con un número.

operador?(p)

Devuelve true si el parámetro es un String con un operador válido.

parentesis?(p)

Devuelve true si el parámetro es un String con un paréntesis.
